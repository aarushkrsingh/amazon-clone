# Amazon Homepage Clone

A static, front-end clone of the Amazon homepage built entirely with HTML5 and CSS3. This project focuses on replicating the layout, UI components, and styling of Amazon's e-commerce interface using modern CSS techniques like Flexbox.

## Features

* **Global Navigation Bar:** Includes the Amazon logo, location selector, a flexible search bar with a dropdown, account sign-in, returns/orders, and a shopping cart icon.
* **Secondary Navigation Panel:** A sub-navigation menu featuring links to "Today's Deals", "Customer Service", and more.
* **Hero Section:** A large hero banner with a promotional message overlay.
* **Product Grid (Shop Section):** A responsive-style flexbox layout displaying various product categories (Clothing, Electronics, Furniture, etc.) in structured card containers.
* **Comprehensive Footer:** A multi-layered footer featuring a "Back to Top" button, categorized links, brand logo, and copyright information.
* **Interactive Elements:** Hover states implemented for navigation items and search bar highlighting.

## Technologies Used

* **HTML5:** Semantic markup for page structure.
* **CSS3:** Custom styling, Flexbox for layout management, and responsive design principles.
